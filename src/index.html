gd-yt

gd-yt is a simple Geometry Dash modification that lets users use songs from YouTube in their custom levels.

This project compiles itself to a DLL file which, after being injected into a Geometry Dash process, can change the inner workings of the game. It does so by utilising a technique called hooking. Whenever certain functions in the game are called the mod diverts the control flow of the main application in order to extend the already existing functionality.

In this case the modification enables users to use YouTube songs in their custom levels. By default players are limited to the tracks uploaded on Newgrounds and manually allowed by the developer. Although it provides creators with an impressive musical repertoire it still doesn't include every single existing song. This modification aims to solve this problem.

To make this project I used the default DLL file template in Visual Studio 2022. It was fully written in the C++ programming language. It utilises many different libraries, including MinHook, gd.h and nlohmann's json.

WokaMeeting

WokaMeeting is a decentralised peer-to-peer online meeting platform.

The application relies mainly on the WebRTC technology, more specifically peer.js library. It is used to establish direct connections between meeting participants without the need of a central server. The user interface was built using Svelte, one of the most loved JavaScript frameworks, and TailwindCSS, in order to make writing CSS faster.

Unlike most solutions, like Microsoft Teams or Zoom, WokaMeeting works entirely client-side. It doesn't need any additional servers since everything is handled on the user's computer. This in turn greatly increases the privacy because all the connections between users are made directly. It also accounts for the decreased latency and improved performance.

WokaMeeting works entirely in a web browser without the need of installing any additional software. It provides many useful functions e.g. screen sharing and text chat. It also works as PWA and its UI is fully responsive.

Chrobry Zastępstwa

Chrobry Zastępstwa is a Facebook Messenger bot which notifies its users of substitutions at Bolesław I the Brave High School in Piotrków Trybunalski.

The bot was entirely written in the Python programming language. As its core framework it uses a web framework Flask and one of its extensions Flask-SQLAlchemy for handling interactions with database. It also utilises pymessenger for communicating with Facebook API and BeautifulSoup4 for parsing data from the school website.

The whole application is split into three parts: homepage, dashboard and the actual bot. The second one makes it able for an administrator to customise its behaviour and configure it according to their needs. The bot itself utilises webhooks to interact with Facebook API and receive messages from users. The app uses Vercel for free hosting and cron-job.org for running periodic cron jobs.

Chrobry Zastępstwa was made in order to automate the boring process of periodically checking current substitutions on the school website. Instead it sends a proper message every time the website's content is changed. It also filters the data so that a user will only see the substitutions they need.
